Kanban.md

VS Code meets Kanban. Transform any Markdown file into a visual board.

Install Extension View on GitHub

Features

Drag & Drop

Flicker-free task management with smooth animations and cross-column preview.

Pure Markdown

Your data stays in .md files. No lock-in, no sync, no cloud.

Real-time Sync

Changes in the board reflect instantly in your Markdown file and vice-versa.

Rich Tasks

Priorities, due dates, checklists, hashtags, and workload indicators.

Quick Start

Create a file ending in .kanban.md and open it with the Kanban view:

# My Board

## To Do
### Task one
### Task two

## In Progress
### Working on this

## Done
### Completed task

Why Kanban.md?

  • No context switching — manage tasks without leaving your editor
  • Version control friendly — it's just Markdown, commit and diff as usual
  • Works offline — no account, no internet required
  • Lightweight — built with React 19 and optimized for performance
